<DESCRIPTION>
I add two targeted enhancements to increase FirstFit pressure without lifting the clique number notably:

1) Cross4 connectors on the deepest spine rounds
- I extend the classic four KT connectors with an optional long-range cross connector, enabled only in the last two spine rounds. This deterministic, capacity-safe addition strengthens cross-scale color coupling while preserving the low-omega spine. It is gated per-round via a new add_cross4 flag passed to apply_round and forwarded to the connector builder.

2) A second, distinct delta2 micro-phase
- After the existing thin micro-phase, I run an additional guarded micro-round with a different set of four fractional windows concentrated near the ends and mid-late sections: [(0.05, 0.15), (0.28, 0.38), (0.60, 0.70), (0.82, 0.92)]. This provides late-stage FF pressure and complements the first micro-phase. For this second micro-phase I also add a single long-range fractional cross connector to further mix colors. All micro additions are budget-bound by CAP and thin-seeded to avoid omega blow-up.

These changes implement recommendations to include explicit cross4 connectors and a second micro-phase with distinct windows, keeping the sequence under 10000 intervals. The edits are minimal and consistent with the existing structure.
